**EzPz** is a powerful, user-friendly management tool designed for **Project Zomboid** dedicated servers. Built with Godot 4 and C#, it streamlines the complex process of setting up, configuring, and managing your server, so you can spend less time in config files and more time surviving the apocalypse.
